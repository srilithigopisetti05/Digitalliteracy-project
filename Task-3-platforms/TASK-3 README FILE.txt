📖 Overview:

This project involves the creation of a Google Form quiz aimed at promoting awareness about digital literacy among students. The quiz includes five thoughtfully designed questions that cover key aspects such as online safety, password security, and responsible digital behaviour.

⚙️ Tools Used:
Google Forms
🧠 Key Features:
Contains exactly 5 questions
Includes both multiple-choice and short-answer formats
Simple and user-friendly design
Focused on real-world digital scenarios
🎯 Objective:

To educate students about the importance of digital literacy and help them identify safe and responsible practices while using digital platforms.
